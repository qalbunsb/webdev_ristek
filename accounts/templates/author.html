About Me

I'm Qalbun Salim Buanaputra, an undergraduate student majoring in Computer Science at University of Indonesia. I'm currently doing my 4th semester and i am looking forward to be a member of RISTEK Fasilkom UI.

Currently, i don't have any professional experiences related to web development, but i have participated in college web projects for final exam. At that time, my group and i made a website with Django Framework named RuangSinggah. RuangSinggah is a mental-health platform where users can make consultations, read articles, and also take a psychology quiz. In RuangSinggah, i contributed on making authentication and registration system for the website, and also creating API endpoints so that the data can be used in RuangSinggah mobile app.

I have experiences in Django and Bootstrap framework, and few experiences in Java Spring Boot. For mobile apps, i have few experiences using Flutter as a front-end framework. I'm looking forward to learn new tools in the future, especially front-end frameworks, such as Javascript React.
